leftasrain
==========

Download songs from `leftasrain`_ from the comfort of your terminal.

Requirements
------------

* Python 3.5
* requests

Installation
------------

::

    $ pip install leftasrain

Usage
-----

To use it, simply execute:

::

    $ leftasrain <song_id>

Where *<song_id>* is a numerical song ID from `leftasrain`_.


.. _leftasrain: http://leftasrain.com/
